# Fonts

## Orkney Font

Place Orkney font files here:
- `Orkney-Regular.woff2`
- `Orkney-Medium.woff2`
- `Orkney-Bold.woff2`

You can download Orkney from a font provider or use a similar geometric sans-serif font as fallback.

If Orkney is not available, the app will fall back to system fonts.

## Arabic Font

Cairo is loaded from Google Fonts automatically for Arabic text.
